# Discord OLLAMA Bot zh-CN

这是本项目的简体中文补充说明文件。

本项目是一个在高中时期制作并整理公开的 Discord Bot 学习项目，主要结合 Discord 服务器管理、本地 Ollama AI、案件报告、用户互动摘要、PC 状态监控、高温保护和个人音乐库功能。

## 项目定位

本仓库主要用于：

- 学习记录
- 作品集展示
- 教学参考
- Discord Bot 与本地 AI 整合示例

除非有特殊原因，本项目未来可能不会进行大规模更新。

## 安全提醒

请不要公开 `.env`、Discord Bot Token、Webhook URL、API Key、密码、`data/`、`logs/`、AI 对话记录、用户资料、案件记录或音乐文件。

如果任何 token、密钥、webhook 或密码被公开，请立即撤销或重新生成。
