# Commands Overview

## Member Commands

- `/status` — Show bot and server status.
- `/my_warnings` — Show your warning status.
- `/my_profile` — Show your interaction summary; optional AI insight.
- `/ai` — Ask the local Ollama model.
- `/ai_long` — Collect long text or attachments before asking AI.
- `@bot message` — Mention-based AI chat.

## Moderation Commands

- `/banword_add`, `/banword_remove`, `/banword_list`
- `/safe_add`, `/safe_remove`, `/safe_list`
- `/case_list`, `/case_resolve`, `/case_ignore`
- `/case_clear`, `/case_clear_test`
- `/warn`, `/calm`, `/remove_warning`, `/punish`
- `/auto_mod`, `/auto_mod_status`

## System Commands

- `/bot_mode`, `/bot_mode_status`, `/bot_guard_set`
- `/summarize_docx`, `/forget`
