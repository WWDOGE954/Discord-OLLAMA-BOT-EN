# コマンド概要

この文書は、各言語向けの簡単なコマンド説明です。完全な内容と実際の動作は、メイン文書 `docs/COMMANDS.md` と実際のコードを優先してください。

## 一般ユーザー向けコマンド

- `/status`：Bot の基本状態を確認します。
- `/ai`：ローカル Ollama AI に質問します。
- `@bot メッセージ`：Bot をメンションして返信を依頼します。
- `/my_warnings`：自分の警告記録を確認します。
- `/my_profile`：自分の簡易プロフィールや対話概要を確認します。

## 管理・モデレーション

- `/warn`：ユーザーに警告記録を追加します。
- `/anger`：ユーザーの怒り値を確認します。
- `/calm`：ユーザーの怒り値を下げる、またはリセットします。
- `/punish`：管理者の判断で処理を実行します。
- `/mod_suggest`：管理判断のための提案を生成します。
- `/remove_warning`：指定した警告記録を削除します。

## ルールと安全フレーズ

- `/banword_add`：禁止語を追加します。
- `/banword_remove`：禁止語を削除します。
- `/banword_list`：禁止語リストを表示します。
- `/safe_add`：安全フレーズまたは許可表現を追加します。
- `/safe_remove`：安全フレーズを削除します。
- `/safe_list`：安全フレーズ一覧を表示します。

## ケースと自動モデレーション

- `/case_ignore`：指定したケースを無視します。
- `/case_resolve`：ケースを処理済みにします。
- `/auto_mod`：自動モデレーション機能を切り替え、または設定します。
- `/auto_mod_status`：自動モデレーションの状態を確認します。

## システム・PC 状態

- `/pc_status`：CPU、RAM、ディスク、ネットワーク、GPU などの状態を確認します。
- `/pc_monitor_start`：状態監視を開始します。
- `/pc_monitor_stop`：状態監視を停止します。
- `/pc_action`：事前定義されたシステム操作を実行します。
- `/bot_mode`：normal、eco、emergency などの Bot モードを切り替えます。
- `/bot_mode_status`：現在の Bot モードを確認します。
- `/bot_guard_set`：温度保護や安全モード関連の設定を行います。

## 音楽機能

- `/music_add`：音楽ファイルまたは音楽項目を追加します。
- `/music_list`：音楽ライブラリを表示します。
- `/music_remove`：音楽項目を削除します。
- `/go_music`：ボイスチャンネルに参加して再生を開始します。
- `/music_queue`：再生キューを確認します。
- `/music_now`：現在再生中の項目を確認します。
- `/music_pause`：一時停止します。
- `/music_resume`：再生を再開します。
- `/music_skip`：現在の項目をスキップします。
- `/music_stop`：再生を停止し、キューを空にします。
- `/music_volume`：音量を調整します。
- `/music_loop`：ループ再生を切り替えます。

## 注意

一部のコマンドには管理者ロール、特定チャンネル、または Discord Bot 権限が必要です。実際に利用できる機能は `.env` 設定、サーバー権限、Discord Developer Portal の設定によって変わります。
